¿Que es un Proyecto Java?

Un proyecto Java podemos considerarlo como una serie de carpetas ordenadas y organizadas
de acuerdo con una lógica para mantener organizado el código. Un proyecto suele constar
de archivos .java, archivos .class y documentación
